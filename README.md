DAA Lab Programs (Week 1 - Week 8)

This repository contains all Design and Analysis of Algorithms (DAA) Lab Programs implemented in C++.

The programs cover important algorithmic techniques such as:

Searching Algorithms
Sorting Algorithms
Divide and Conquer
Greedy Algorithms
Graph Algorithms
Dynamic Programming

All codes are written in a simple and beginner-friendly way for lab practicals and academic purposes.

Linear Search
Binary Search
Jump Search

Count pairs with given difference

Duplicate Elements Detection
Week 4
